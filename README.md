## Usage

1. Clone or fork this repo

   ```shell
   $ git clone https://github.com/MohsenDastaran/portfolio
   ```

2. Install all the dependencies

   ```shell
   $ yarn
   ```

   or

   ```
   $ npm install
   ```

3. Run the local server!
   ```shell
   $ yarn dev
   ```
   or
   ```shell
   $ npm run dev
   ```
